hello web designer

My passion for Web Design came to me at the beginning of the 2018 school year, when I participated in a UCONN/ECE class for Digital Media. I quickly learned how to code a website, which is now the portfolio in which you are currently viewing. I have always had an eye for designing things-- when I was little I was convinced I was going to be an interior designer because I was constantly rearranging my bedroom and imagining new ways to make it come to life. I feel it's the same idea with Web Design. I get to choose what is displayed and in what fashion, and believe that my voice finds a way to be known within it.
